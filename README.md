# KMZ → GPX конвертер

Веб-приложение для конвертации KMZ/KML в GPX прямо в браузере — без установки, без сервера. Работает на десктопе и смартфоне, можно установить как PWA.

**Демо:** https://agran.github.io/kmz2gpx/

## Возможности

- Загрузка KMZ (архив) или обычного KML файла — можно загружать несколько файлов подряд, точки/треки добавляются к уже имеющимся
- Разбор точек (Point), треков (LineString) и Google `gx:Track`
- Ручное добавление точек — формой (название + широта/долгота) или кликом по карте
- Отображение всех треков и точек на карте OpenStreetMap
  - выбранное — синим, невыбранное — серым
  - можно выбирать/снимать выбор прямо кликом по карте, синхронно со списком
- Выбор нужных треков и точек чекбоксами в списках
- Копирование в буфер обмена или скачивание `.gpx`:
  - всё (треки + точки)
  - только выбранные треки
  - все точки
  - только выбранные точки
- Режим «точки как маршрут» — экспортирует точки как трек (`<trk>`) вместо waypoints, для приложений (например, Zepp), которые не импортируют одиночные точки; точки при этом дополнительно включаются в GPX и как `<wpt>`. На карте при этом показывается линия-предпросмотр будущего маршрута.
- Устанавливается как приложение (PWA) на Android и iOS, работает офлайн после первого открытия

## Использование

1. Открыть [демо-страницу](https://agran.github.io/kmz2gpx/)
2. Выбрать KMZ/KML файл
3. Отметить нужные треки/точки в списках или на карте
4. Скопировать или скачать результат в формате GPX

## Локальный запуск

Статические файлы, сборка не требуется. Достаточно раздать папку любым http-сервером, например:

```
python -m http.server 8877
```

и открыть `http://localhost:8877/index.html`.
